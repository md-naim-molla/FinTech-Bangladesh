Policy Recommendations
App Stability
Mandatory Pre-Release Regression Testing
Major update drops correlate with sudden spikes in 1-star reviews (e.g., Jan 2024 dip to 2.10 avg). Enforce device-matrix testing across Android versions 9–14 and budget OEMs (Tecno, Infinix, Symphony) before every release.
High Priority
Authentication
Biometric Login as Default Option
3,427 reviews mention login friction. Users repeatedly request fingerprint/face ID as the default path. Mandate biometric auth rollout across all supported devices and invest in fallback OTP that allows manual entry.
High Priority
Transaction Reliability
Failed Transaction SLA Policy
5,957 payment-related reviews — the most discussed topic. 43% are negative. Introduce a 24-hour automatic refund SLA for failed/deducted-but-undelivered transactions, and display real-time server status within the app.
High Priority
Customer Support
Upgrade Live Support Experience
The helpline and live chat are frequent complaints — sessions refresh mid-chat, calls disconnect while navigating IVRS menus. Implement persistent chat sessions, and introduce a callback feature to eliminate wait-time frustration.
Medium Priority
Pricing Transparency
Fee Disclosure & Reduction Strategy
1,875 charge-related reviews — 55% negative. Users compare service charges to competitors and find bKash expensive. Introduce in-app fee calculators before confirming transactions, and consider a tiered loyalty discount programme.
Medium Priority
Security & Trust
Proactive Fraud Prevention Communication
394 security-related reviews with nearly 50% negative. Users report unauthorized access and fraud. Launch periodic in-app security tip campaigns, add transaction anomaly alerts, and create a dedicated fraud reporting fast-track.
Medium Priority
UX Improvements
UI Modernisation Roadmap
1,260 UI/UX reviews — 81% are actually positive or neutral, showing appetite for improvement. Users request better font sizing, cleaner navigation, and balance history visibility. A quarterly design sprint cycle would sustain momentum.
Lower Priority
Monitoring
Real-Time Sentiment Monitoring Dashboard
The Jan 2024 rating collapse (2.10 avg) could have been caught within hours with automated alerting. Deploy a CI/CD-integrated Play Store review monitor that triggers P1 alerts when 7-day average drops below 3.0.
